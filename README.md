## Nome del Progetto / Project Name

Controllo e identificazione pendolo inverso su cart / Control-Identification-Reverse-Pendulum-on-Cart

---

### Scopo del Progetto / Project Aim

(IT) : Identificare il sistema instabile e non lineare. Poi controllarlo con tecnica MPC

(EN) : Identify the instable and non-linear system. Then control it with MPC control

---

### Linguaggi e tecnologie / Languages & Technologies

- Matlab + Simulink

---

### Utilizzo / Usage

(IT) : Apri in matlab il file livescript .mlx d'interesse e premi su run

(EN) : Open in matlab the desidered livescript file .mlx and then click on 'run'

---

### Struttura / Structure

- *.mlx                (codice sorgente / source code)
- controller_savings   (salvataggio MPC / MPC saving)
- functions            (funzioni matlab / matlab functions)
- id_models            (salvataggio modelli / models savings)
- id_sessions          (sessioni d'identificazione / identification sessions)
- simulink_models      (circuiti simulink / simulink models)
